Japan Clickable Map Silverlight は、都道府県ごとのクリックが可能な日本地図のテンプレートです。
地図の部分にはインラインSVGを使用しているため、インラインSVGをサポートしているブラウザでお使いください。

Japan Clickable Map Silverlight is a template of Japanese map in which you can click every prefecture.
It's made using Inline SVG.
Use the browser which supported Inline SVG.

(DEMO)
http://www.project0884.com/github/japan-clickable-map-svg/demo/demo1.html
